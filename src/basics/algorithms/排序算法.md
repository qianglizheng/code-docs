## 1. 冒泡排序

算法思想：
- 将 `0` 到 `n-1` 中最大的数冒泡到 `n-1` 位置
- 将 `0` 到 `n-2` 中最大的数冒泡到 `n-2` 位置

核心：长度为 `n` 的数组，需要冒泡 `n - 1` 次

举个例子：数组 `3 2 1`
- 第一步将3冒泡到2位置 `2 1 3`
- 第二步将2冒泡到1位置 `1 2 3`
```java
public static void main(String[] args){
    int[] arr = {3, 2, 1};
    int n = arr.length;
    for(int end = n - 1; end > 0; end--){
        for(int i = 0; i < end; i++){
            if(arr[i] > arr[i + 1]){
                swap(arr, i, i + 1);
            }
        }
    }
}

public static void swap(int[] arr, int i ,int j){
    int temp = arr[i];
    arr[i] = arr[j];
    arr[j] = temp; 
}
```
